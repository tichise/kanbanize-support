カンバン型リマインダーアプリ

2024.1.18 Kanbanize リリース 開発ブログ

主な特徴

カンバン型レイアウト

タスクを「To Do」「In Progress」「Done」などのカテゴリーで整理し、進捗をリアルタイムで更新。

iOS標準リマインダーとの連携

EventKitを活用し、iOSリマインダーの情報を簡単に取り込んで、タスク管理を一元化。

ドラッグ＆ドロップ機能

タスクの進行状況をドラッグ＆ドロップで簡単に変更でき、効率的に管理。

アプリ名の由来

「Kanbanize」という名前には、カンバン形式のUIを取り入れていることが由来しています。「Kanban」は、日本の製造業で使用されるタスクの可視化と進捗管理のシステムを指します。

また、「付箋を使ったカンバン」の持つ自由度の高さをシステムに落とし込みたいという考えも、この名前に込められています。現在のUIは「まず最初の形」として設計されており、今後、より柔軟性と自由度の高いUIを提供することで、個々のワークスタイルやタスク管理のニーズに対応していく予定です。

機能追加予定

ワークフローの統合

GitHub ActionsとGitHub Projectsの連携のように、タスクを実行可能なワークフローと結びつける機能をリマインダーアプリでも再現します。

AIエージェントによる自動化

タスクの自動振り分けや進捗更新など、AIエージェントを活用した作業自動化機能を開発中です。
